\chapter{Comparação das novidades com os artistas conhecidos}\label{cap:comparacao}

Até este momento trabalhamos apenas com novidades. Porém, é possível que as preferências dos ouvintes pelas novidades sejam as mesmas que pelos artistas conhecidos? Caso esta hipótese seja verdade, o comportamento dos ouvintes para estes dois âmbitos, novidades e não-novidades, seriam similares. Para responder esta pergunta expandimos nossos experimentos para englobar também os artistas conhecidos, tentando responder a seguinte pergunta de pesquisa: \textit{As relações entre as preferências dos ouvintes e os aspectos das novidades são as mesmas que as relações entre as preferências dos ouvintes e os aspectos dos artistas já conhecidos?}

\section{Seleção de sujeitos}
Para esta análise, utilizamos os mesmos 11.989 sujeitos das análises anteriores e aplicamos um filtro, para selecionar os sujeitos que foram expostos a um número mínimo de artistas conhecidos que permitisse a investigação de relações entre os aspectos destes artistas e as preferências. Filtro similar foi descrito na Seção \ref{sec:Filtros} para a seleção de sujeitos que foram expostos a um número mínimo de novidades.

Desta maneira, foram excluídos os usuários que escutaram menos de 10 artistas conhecidos no Período de Observação. Após esta exclusão ficamos com uma amostra de 10.207 sujeitos para os experimentos.

\section{Características dos artistas conhecidos}\label{sec:comp_carac}
		
Como dito na Seção \ref{sec:timeline}, os artistas conhecidos são os artistas escutados no Período de Observação que já foram escutados previamente no Histórico Inicial do ouvinte. Para eles foram calculadas 3 das 4 métricas que foram calculadas para os artistas com novidade (Capítulo \ref{cap:dados}): familiaridade e popularidade (aspectos) e total de atenção (preferência).

Para calcular as relações entre os aspectos dos artistas conhecidos e as preferências pelos ouvintes, utilizamos a mesma metodologia descrita no Capítulo \ref{cap:pref}. Utilizamos o método de correlação não-paramétrico de Spearman entre cada par de aspecto / preferência.

\section{Comparação entre relações das preferências e aspectos das novidades e dos artistas conhecidos}\label{sec:comp_pref}
Para guiar nossos experimentos, decidimos especificar a pergunta de pesquisa: \textit{As relações entre as preferências dos ouvintes e os aspectos das novidades são as mesmas que as relações entre as preferências dos ouvintes e os aspectos dos artistas já conhecidos?}, que é mais genérica. Assim, tentamos responder duas perguntas de pesquisa, uma envolvendo a diferença destas relações e outra envolvendo a correlação destas relações.



\subsection{Diferença das relações entre preferências e aspectos das novidades e dos artistas conhecidos} \label{subsec:diferenca_relacoes}

A primeira pergunta levantada foi: \textit{As relações entre as preferências dos ouvintes e os aspectos das novidades são significantemente diferentes das relações entre as preferências dos ouvintes e os aspectos dos artistas já conhecidos?}	Se as relações comparadas forem significantemente iguais, poderíamos estender os resultados das novidades para o comportamento geral do ouvinte. Se forem diferentes, corroboraremos a importância da análise separada de itens com novidades e itens conhecidos.




Para cada um dos 10.207 sujeitos foram calculados os valores das correlações entre a familiariade e o total de atenção e entre a popularidade e o total de atenção, tanto para os artistas com novidade quanto para os artistas conhecidos. Os detalhes destes cálculos foram apresentados no Capítulo \ref{cap:pref} e na Seção \ref{sec:comp_carac}. Desta maneira cada ouvinte possui pares de correlações (Tabela \ref{tab:correlacoes_nov_con}), onde um valor do par é referente aos artistas com novidade e o outro aos artistas conhecidos. Por exemplo, cada ouvinte possui um valor de correlação entre a familiaridade e o total de atenção para as novidades e um valor de correlação entre a familiaridade e o total de atenção para os artistas conhecidos. 

Desta maneira, para responder a pergunta exposta no parágrafo anterior, utilizamos um teste-T pareado, para cada par de correlação novidade / artista conhecido correspondente de todos os ouvintes. O intuito desta análise é descobrir se, por exemplo, o valor da correlação entre a familiaridade e o total de atenção para as novidades e para os artistas conhecidos são significantemente diferentes.

\subsubsection{Análise das correlações envolvendo familiaridade}
Um fato que podemos observar da tabela é que o intervalo de confiança, com 95\% de confiança, para as diferenças das correlações entre familiaridade e atenção total para artistas com novidade e artistas conhecidos é [-0,14 -0,13], apontando que as correlações entre a preferência e a familiaridade para os artistas conhecidos  são maiores que para os artistas com novidade. Isto fica evidenciado na Figura \ref{fig:fam_nov_con}. Na figura, para cada ouvinte  (eixo horizontal) plotamos os valores das correlações da preferência com a familiaridade para os artistas conhecidos e artistas com novdade. Os ouvintes foram ordenados pelo valor da diferença entre as correlações. Pode-se perceber que a proporção de ouvintes  com correlação para artistas conhecidos maior que para artistas com novidades é bem maior que o contrário.

Apesar desse resultado, não sabemos se no geral ambas as correlações para o ouvinte são positivas, negativas ou de sinais diferentes. Podemos tirar conclusões diferentes a partir do sinal das correlações comparadas. Para investigar isso, plotamos a Figura \ref{fig:fam_divid}. A figura foi separada em 6 partes, de acordo com o sinal das correlações e com o sinal da diferença. Podemos perceber que, quando a correlação para os artistas conhecidos é maior que para os artistas com novidade (linha 1), a maior parte dos ouvintes ou possuem ambas as correlações positivas (gráfico da posição (1,3)) ou uma positiva e outra negativa (gráfico da posição (1,2)), tendo uma pequena porção dos ouvintes possuindo ambas as correlações negativas.

Podemos concluir que, dos ouvintes com correlação entre a familiaridade e atenção total maior para artistas conhecidos que para artistas com novidade, parte possuem uma tendência maior de gostar de artistas conhecidos familares que artistas com novidades familiares, parte possuem uma tendência em gostar de artistas conhecidos familiares junto com uma tendência em gostar de artistas com novidade não-familiares, enquanto uma pequena parcela têm tendência maior em gostar de artistas com novidade não-familiares que artistas conhecidos não-familiares.

\subsubsection{Análise das correlações envolvendo popularidade}

Já o intervalo de confiança, com 95\% de confiança, para as diferenças das correlações entre popularidade e atenção total para artistas com novidade e artistas conhecidos é [-0,04 -0,03] (Tabela \ref{tab:testeT}). Isto também sugere que as preferências pelos artistas mais populares conhecidos são maiores que pelos artistas mais populares com novidade, apesar de que este intervalo de confiança está mais próximo de zero que o intervalo de confiança anterior. Na Figura \ref{fig:pop_nov_con} podemos notar que a proporção de ouvintes com correlação entre popularidade e atenção total para artistas conhecidos maior que para artistas com novidade é um pouco maior que o inverso.

Com essas duas análises podemos concluir que as relações entre os aspectos e as preferências para os artistas com novidade e artistas conhecidos são diferentes, existindo nos dados diferentes tipos de sujeitos a partir das características destas duas relações.

\subsection{Correlação das relaçãoes entre preferências e aspectos das novidades e dos artistas conhecidos}

Já a segunda pergunta de pesquisa, \textit{Existe correlação entre as relações das preferências pelos aspectos dos artistas com novidade e as relações das preferências pelos aspectos dos artistas conhecidos?}
Próximo passo foi verificar se existiam correlações entre as relações calculadas para os artistas com novidade e para os artistas conhecidos. Existe a possibilidade de, por exemplo, quanto maior a correlação de um ouvinte entre familiaridade e atenção total para artistas com novidade, maior a correlação entre a familiaridade e atenção total para artistas conhecidos? Para responder a pergunta de pesquisa, utilizamos o método de correlação não-paramétrico de Spearman.


\begin{table} [htpb]
 \begin{center}
  
 \begin{tabular}{ |c|c|c|c| }
  \hline
  Novidades X Artistas conhecidos & cor(fam., aten. tot.) & cor(pop. , aten. tot.) \\
  \hline
  cor(familiaridade , atenção total)  & 0,05  & -0.02 \\
  cor(popularidade , atenção total)   &  -0.04 & 0,09 \\
    \hline
      \end{tabular}
      \caption{Correlação (Coeficiente de Spearman) entre correlações calculadas para novidades (linhas) e artistas conhecidos (colunas)}\label{tab:comp_cor}
\end{center}
\end{table}
  
 Como podemos observar na Tabela \ref{tab:comp_cor}, a correlação é baixa em todos os casos. Não podemos afirmar, por exemplo, que dado dois ouvintes, se um possuir maior correlação entre familiaridade e período de atenção para novidades que o outro ouvinte ele terá maior chance de ter maior correlação entre familiaridade e período de atenção para artistas conhecidos, e vice-versa. 
 
 Esta falta de correlação geral pode ser ocasionada pela diversidade de comportamentos para novidades e artistas conhecidos dos ouvintes. Este resultado, junto com os diferentes tipos de ouvintes analisados na Subseção \ref{subsec:diferenca_relacoes}, nos fez seguir a análise feita no Capítulo \ref{sec:grupos}, onde criamos perfis de ouvintes baseados nas relações entre preferências e os aspectos das novidades (Capitulo \ref{sec:grupos}). Próximo capítulo abordará a criação de perfis envolvendo as relações entre aspectos e preferências para as novidades como também as relações para artistas conhecidos. 


\section{Grupos de ouvintes baseados na diferença das relações entre preferências e aspectos das novidades e dos artistas conhecidos}

Seguindo a metodologia dos Capítulos \ref{sec:grupos} tentamos encontrar perfis de ouvintes, com o intuito de responder a seguinte pergunta: \textit{Quais os diferentes perfis de ouvintes baseados nas relações entre os aspectos e preferências para as novidades e as relações entre aspectos e preferências para artistas conhecidos?}

Para isso utilizamos 7 métricas, 5 utilizadas no Capítulo \ref{sec:grupos} junto com correlação entre a familiaridade dos artistas conhecidos e a atenção total e a correlação entre a popularidade dos artistas conhecidos e a atenção total. As 7 métricas estão listadas a seguir:
\begin{enumerate}
 
  \item \textit{Correlação entre a familiaridade das novidades e a atenção total dedicada a elas durante o experimento.}
  \item \textit{Correlação entre popularidade das novidades e a atenção total dedicada a elas durante o experimento.}

  \item \textit{Ecleticidade}

  \item \textit{Popularidade média dos artistas do perfil do ouvinte} 

  \item \textit{Proporção de novidades escutadas pelo ouvinte no Período de Observação} 

  \item \textit{Correlação entre a familiaridade dos artistas conhecidos e a atenção total dedicada a eles durante o experimento.}
  \item \textit{Correlação entre popularidade dos artistas conhecidos e a atenção total dedicada a eles durante o experimento.}

\end{enumerate}
